# Bayesian Methods in Nuclear Physics
This repository contains mini projects for the **Bayesian tutorial** session at the 4th PhyNuBe school: Innovative Technologies, Theories, and Methodologies in Nuclear Physics, delivered by Prof. Christian Forssén on May 18-22, 2026 at the CNRS conference center  "Paul Langevin" in Aussois, France.

- Advanced projects for the tutorial session are available in the [Projects](Projects) directory.

The lectures are accompanied by a Jupyter Book. This book covers several topics that are not included in the school lectures. The book can be accessed at [https://nucleartalent.github.io/LFD_for_Physicists/](https://nucleartalent.github.io/LFD_for_Physicists/).
